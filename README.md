# ScipioFX
[![](https://jitpack.io/v/ScipioAM/scipio-fx.svg)](https://jitpack.io/#ScipioAM/scipio-fx)
#### 介绍
JavaFX轻量级框架 (Java 17+)

#### 使用说明

1.加入JitPack的仓库地址
```xml
<repositories>
    <repository>
        <id>jitpack.io</id>
        <url>https://jitpack.io</url>
    </repository>
</repositories>
```
2.maven依赖引入
```xml
<dependency>
    <groupId>com.github.ScipioAM</groupId>
    <artifactId>scipio-fx</artifactId>
    <version>latest</version>
</dependency>
```
